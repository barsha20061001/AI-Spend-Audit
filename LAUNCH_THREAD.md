# AI Spend Audit Launch Thread

Most startups are overspending on AI tools without realizing it.

We built AI Spend Audit to help teams quickly identify:
- overpriced plans
- duplicate subscriptions
- unnecessary enterprise tiers
- cheaper alternatives

The tool analyzes:
- ChatGPT
- Claude
- Gemini
- Cursor
- GitHub Copilot
- OpenAI API usage
- Anthropic API usage

It gives practical recommendations like:
- downgrade opportunities
- API optimization ideas
- lower-cost alternative tools
- high-spend anomaly detection

The goal is not “AI opinions.”

The goal is defensible cost optimization backed by pricing data and clear reasoning.

Features:
- shareable audit reports
- transactional email confirmation
- lead capture backend
- real pricing references
- mobile-friendly UI

Built with:
- Next.js
- TypeScript
- Tailwind
- Supabase
- Resend
- Vercel
